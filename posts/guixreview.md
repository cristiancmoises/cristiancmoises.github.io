### What is Gnu Guix?
A amazing operating system! 
- Different of others GNU/LINUX
- GUIX package are fast and stable
- Integration with Emacs
- Run fast on all machines
- Fast Init System
- Great control of your system versions
- Default: kernel hurd
- All configuration in one file!
